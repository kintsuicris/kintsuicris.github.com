writing skills in "Markdown"

Posted on 2016-02-04   |   In Note

Add “link”

1. website link
This is an example -> Google
The structure of it is [website Name]+(link “description”).
In the structure, “+” means “add the following element immediatly.
Thus, the Google link is writen in -> [Google]+(http://google.com “Google”).
2. in-site link
This is an example -> About
This About page is inside cjian.info, and the structure of it is:
[in-site link name]+(/link/)
Thus, the About link is writen in -> [About]+(/about/).
